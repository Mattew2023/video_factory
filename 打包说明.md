# 打包说明

发给别人时必须把 exe 和 ffmpeg 文件夹一起发送。

分发目录结构：

```text
抖音合并压缩工具.exe
ffmpeg/
  ffmpeg.exe
  ffprobe.exe
```

程序启动时会先检测 exe 所在目录下的 `ffmpeg/ffmpeg.exe` 和 `ffmpeg/ffprobe.exe`。如果这两个文件不存在，才会检测系统 PATH 里的 ffmpeg 和 ffprobe。

如果两处都找不到，程序会提示：

```text
未检测到 ffmpeg，请把 ffmpeg.exe 和 ffprobe.exe 放到软件目录下的 ffmpeg 文件夹中。
```

注意：这里的 `ffmpeg` 文件夹要和 exe 放在同一级目录，不是放到 exe 里面，也不是放到 `ffmpeg/bin` 里面。
