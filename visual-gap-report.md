# /qianchuan-dashboard 视觉差距报告

## 一、整体相似度判断

当前页面与目标页面的整体相似度评分：**82 / 100**。

给分理由：

- 页面类型已经对齐：当前页已经不是旧 `/dashboard` 的罗盘复盘结构，而是「顶部导航 + 公告条 + 核心指标横卡 + 左侧趋势图 + 右侧三张经营分析卡片」的千川大盘结构。
- 主要模块完整：顶部导航、核心指标、整体趋势、成交渠道构成、直播间核心漏斗、直播实时评论都已经出现。
- 视觉氛围接近：深色背景、蓝色指标卡、圆角卡片、右侧分析栏、趋势图面积填充都接近目标图。
- 仍有明显差距：趋势图区域比例偏大，底部事件时间线在当前截图中几乎不可见；右侧卡片的内部排版与目标仍有差异；顶部 logo、公告条、按钮和光晕细节还不够像；部分字号、间距、卡片高度与目标图不一致。

当前版本已经完成「结构拟合」，但还没有完成「像素级视觉拟合」。

## 二、逐模块对比

### 1. 顶部导航区

**高度**

- 目标图顶部导航区高度较克制，导航栏和公告条之间的层级清楚。
- 当前页顶部导航高度接近，但整体略显厚重，品牌区和右侧信息占用的垂直空间稍大。

**logo 位置**

- 目标图左侧「抖音电商 / 巨量千川」logo 靠左且贴近顶部视觉基线。
- 当前页 logo 位置基本正确，但「巨量千川」图形标识不像目标图中的三段柱形标识，当前更像简单白色图形块。

**公告条位置**

- 目标图公告条位于顶部导航下方，横向贯穿主内容区域，右侧有分页和关闭。
- 当前页公告条位置接近，但按钮「查看详情」「我知道了」更靠右，目标图中它们更贴近公告文字；当前公告条整体文字密度略低。

**店铺信息位置**

- 目标图店铺信息在顶部右侧，紧凑排列在直播状态前。
- 当前页位置接近，但整体更靠左一些，店铺头像和文字间距略大。

**刷新数据 / 全屏按钮位置**

- 目标图刷新数据和全屏按钮贴近右上角，按钮尺寸较小、胶囊感更强。
- 当前页按钮位置正确，但按钮宽度和高度略大，视觉重量偏重。

**背景色和光晕**

- 目标图顶部背景有较明显的红、绿、蓝柔和光晕，并融入黑色背景。
- 当前页有光晕，但分布更平均，红色区域和中心绿色光感不如目标明显，顶部更偏整片深蓝黑。

### 2. 顶部核心指标区

**卡片宽度和高度**

- 目标图核心指标卡从左侧主内容区延伸到右侧栏前，宽度非常接近当前页。
- 当前页卡片高度略低或内容更居中，目标图顶部指标区在纵向上更舒展。

**指标数量**

- 两者都包含 8 个核心指标，数量已经对齐。

**指标排列**

- 目标图上排突出「整体消耗、净成交 ROI、净成交金额」，下排分布 GPM、观看成交转化率、实时在线人数、曝光观看率、直播间整体观看人数。
- 当前页排列与目标基本一致，但上排三项整体偏左，下排指标间距略均匀，目标图中左右留白更自然。

**数字字号**

- 目标图主数字非常大，尤其「338.15」「21.44」「7,250.00」更有压迫感。
- 当前页数字已接近，但主指标字号仍略小，字重和发光感也稍弱。

**标签字号**

- 目标图标签字号小且细，带虚线下划线。
- 当前页标签样式接近，但部分标签显得略亮、略大，层级不如目标细腻。

**蓝色渐变背景**

- 目标图蓝色渐变左侧带青绿色暗光，中心高亮明显，右侧收暗。
- 当前页渐变方向和颜色接近，但整体更平、更亮，目标图的深浅层次更强。

**与目标图差距**

- 当前核心指标区已经是相似度较高的模块，主要差距在字号、光晕、留白和指标横向位置。

### 3. 左侧主趋势图

**卡片位置**

- 目标图趋势卡位于核心指标区下方，顶部留白约一条窄间距。
- 当前页趋势卡位置接近，但整体上沿略高，卡片下方被截图底部截断得更明显。

**卡片宽高**

- 目标图趋势卡高度足够容纳图表、时间轴、投放动作、调控动作和底部拖动条。
- 当前页趋势卡中的图表区域过高，底部事件时间线没有完整露出，这是当前最明显的差距之一。

**Tab 样式**

- 目标图 Tab 是白色激活胶囊 + 灰蓝未激活胶囊，尺寸较小。
- 当前页 Tab 样式接近，但按钮更厚，间距略大。

**指标开关**

- 目标图指标开关较紧凑，圆点较粗，文字更小。
- 当前页指标开关整体接近，但占用横向空间偏大，最后一个指标在当前截图里靠右截断感明显。

**5分钟粒度按钮**

- 目标图按钮位于趋势区右上方，靠近筛选图标，尺寸较小。
- 当前页位置正确，但整体偏大、偏圆，和目标按钮的轻量感有差距。

**筛选按钮**

- 目标图筛选按钮是圆形图标，靠右贴近卡片边缘。
- 当前页有圆形筛选按钮，但图标形态不够像，位置略偏内。

**坐标轴**

- 目标图左轴最大值约 800，右轴最大值约 18，横向网格和标签比较清晰。
- 当前页左轴最大值显示到 900，右轴 18，对齐度尚可，但左轴比例与目标不完全一致。

**曲线形态**

- 目标图蓝色和青色曲线走势差异明显，蓝线有多个尖峰，青线更平滑但仍有起伏。
- 当前页两条曲线整体贴得更近，且面积填充较厚，蓝线尖峰差异不如目标明显。

**面积填充**

- 目标图面积填充更克制，透明度较低，主要衬托曲线。
- 当前页面积填充更重，占据视觉中心，导致图表看起来偏满。

**底部事件时间线**

- 目标图底部清楚显示「投放动作」「调控动作」、事件点、拖动条和「+ 新建调控」。
- 当前截图中底部事件时间线几乎没有露出，属于高优先级问题。

### 4. 右侧成交渠道构成

**卡片位置和大小**

- 目标图右侧第一张卡片位于核心指标区同一高度，尺寸较紧凑。
- 当前页位置和大小基本接近，但卡片顶部与指标卡的垂直对齐略有差异。

**环形图大小**

- 目标图环形图较大，位于卡片左侧偏中。
- 当前页环形图大小接近，但略小，中心文字略大。

**图例位置**

- 目标图图例在环形图右侧，排列紧凑。
- 当前页图例位置正确，但渠道名称有截断，目标图中截断也存在，但当前文字密度更低。

**渠道占比排列**

- 目标图占比数字右对齐，行距紧凑。
- 当前页基本对齐，但行距略大，占比数字的视觉强调略弱。

**是否接近目标**

- 该模块相似度较高，主要差距在环形图比例、卡片内边距和图例密度。

### 5. 右侧直播间核心漏斗

**卡片位置和大小**

- 目标图第二张卡片紧贴第一张卡片下方，纵向高度约 210px。
- 当前页位置接近，但卡片内部视觉更松，目标图更像紧凑的数据图表。

**漏斗条样式**

- 目标图漏斗条有明显的右侧尖角或箭头感，层级向下收窄。
- 当前页使用圆角横条，收窄存在，但箭头感不足。

**层级关系**

- 目标图层级关系通过右侧连线和箭头更清楚。
- 当前页层级主要依靠条宽变化和右侧文字，转化路径表达弱一些。

**转化率箭头**

- 目标图转化率与箭头/连线绑定，能看出每一层转化关系。
- 当前页只有短横线和数字，箭头结构不足。

**数据标签**

- 目标图每层名称和数值都比较紧凑，数值右侧对齐。
- 当前页文本较拥挤时会截断，数值还原度可以，但标签密度与目标不同。

### 6. 右侧直播实时评论

**卡片位置和高度**

- 目标图第三张卡片从右侧下半区一直延伸到底部，留有较大空白。
- 当前页高度接近，但评论列表填充更满，目标图评论条数量更少、气泡更短。

**评论条样式**

- 目标图评论更像小胶囊气泡，宽度根据内容变化。
- 当前页评论条是整行宽卡片，块感更重。

**用户名颜色**

- 目标图用户名使用青绿色，较突出。
- 当前页用户名颜色接近，但因为整条评论背景更重，用户名没有目标图那么轻巧。

**标签样式**

- 目标图标签偏弱，更多像右侧按钮或小标记。
- 当前页标签统一放在每条评论右侧，识别清楚，但比目标更规整、更像列表组件。

**内容密度**

- 目标图内容密度低，评论之间留白更多。
- 当前页内容密度偏高，列表感强于实时弹幕/评论气泡感。

## 三、最影响像不像的前 10 个问题

### 1. 趋势图底部事件时间线没有完整露出

- 问题描述：目标图底部清楚展示投放动作、调控动作、事件点、拖动条和「+ 新建调控」，当前截图中底部几乎被图表区域挤出。
- 影响等级：高
- 应修改文件：`qianchuan.css`、`qianchuanChart.js`
- 修改建议：压缩主绘图区高度，增加底部时间线保留区；调整 canvas 内部 `bottom` padding，使事件时间线稳定显示。

### 2. 主趋势图曲线和面积填充过满

- 问题描述：当前图表面积填充占比太重，两条曲线靠得较近；目标图蓝线和青线差异更明显，面积填充更克制。
- 影响等级：高
- 应修改文件：`qianchuanChart.js`、`qianchuanData.js`
- 修改建议：降低面积填充透明度，调整趋势数据让两条线分离度更接近目标，增加蓝线尖峰和青线平滑走势差异。

### 3. 趋势卡高度和内部比例不准

- 问题描述：当前趋势卡里图表区域过大，导致底部结构不足；目标图图表、时间轴、事件行、拖动条比例更均衡。
- 影响等级：高
- 应修改文件：`qianchuan.css`
- 修改建议：重新分配趋势卡内部网格高度，明确 `header / legend / chart / timeline` 的比例。

### 4. 右侧漏斗缺少目标图的箭头和连线关系

- 问题描述：目标漏斗通过箭头和右侧连线表达层级转化，当前仅用横条和短线，结构感弱。
- 影响等级：高
- 应修改文件：`qianchuan.css`、`qianchuanDashboard.js`
- 修改建议：为漏斗条增加右侧箭头造型和纵向连线层，转化率放到对应连线旁。

### 5. 顶部核心指标主数字不够接近目标

- 问题描述：当前主指标数字已经大，但目标图更粗、更亮，主次层级更强。
- 影响等级：中
- 应修改文件：`qianchuan.css`
- 修改建议：略增大主指标字号和字重，降低次级指标视觉重量，微调上排三项位置。

### 6. 顶部品牌 logo 还原度不足

- 问题描述：「巨量千川」图形标识不像目标图中的三段柱形 logo，影响第一眼识别。
- 影响等级：中
- 应修改文件：`qianchuan.css`、`qianchuanDashboard.js`
- 修改建议：重做千川 logo 标识结构，使用三段倾斜柱形块，更贴近目标。

### 7. 顶部背景光晕不够像目标图

- 问题描述：目标顶部有明显红、绿、蓝三段光晕，当前光晕更平均，中心绿色和左侧红色不够明显。
- 影响等级：中
- 应修改文件：`qianchuan.css`
- 修改建议：加强顶部左侧红色光晕、中部绿色光晕和右侧蓝色光晕的位置与透明度。

### 8. 公告条内部布局略偏

- 问题描述：目标公告按钮更贴近文字，分页和关闭更靠右；当前按钮排布偏机械，右侧元素略拥挤。
- 影响等级：中
- 应修改文件：`qianchuan.css`、`qianchuanDashboard.js`
- 修改建议：调整公告条为「徽标 + 文案 + 文案按钮 + 弹性空白 + 分页 + 关闭」布局。

### 9. 直播实时评论太像列表，不像目标气泡

- 问题描述：目标评论是短气泡式信息流，当前是等宽列表卡片，块感偏重。
- 影响等级：中
- 应修改文件：`qianchuan.css`、`qianchuanDashboard.js`
- 修改建议：评论项改为内容自适应宽度的小气泡，减少背景块面积，降低行高和内边距。

### 10. 右侧成交渠道构成图例密度和环形图比例仍有差距

- 问题描述：当前图例行距略大，环形图略小，中心文字略突出；目标图更紧凑。
- 影响等级：低
- 应修改文件：`qianchuan.css`、`qianchuanChart.js`
- 修改建议：略增大环形图半径，缩小中心文字，压缩图例行距并强化占比右对齐。

## 四、下一轮修改优先级

### P0：必须先改

- 修复趋势图底部事件时间线不可见的问题。
- 调整趋势卡内部比例，让图表、事件行、拖动条都完整出现。
- 降低趋势图面积填充重量，调整曲线形态，让两条曲线更接近目标。
- 重做核心漏斗的箭头和转化率连线。

### P1：第二轮改

- 微调顶部核心指标区的字号、字重、横向位置和渐变层次。
- 调整右侧三张卡片的内边距、图表比例和内容密度。
- 优化顶部导航区 logo、公告条、店铺信息、刷新和全屏按钮的精确位置。
- 加强顶部背景光晕，贴近目标图的红、绿、蓝分布。

### P2：最后微调

- 调整指标开关圆点、文字大小和间距。
- 微调坐标轴标签、网格线透明度、图例颜色。
- 调整评论气泡圆角、标签弱化、用户名颜色。
- 统一卡片圆角、描边、阴影和边缘高光。
- 校准渠道构成环形图的扇区比例和图例行距。

## 总结

当前 `/qianchuan-dashboard` 第一版已经完成目标页面的结构复刻，整体已经比较像目标图。下一轮不应优先新增功能，而应优先处理趋势图比例、事件时间线、漏斗箭头结构和顶部指标视觉层级。这几项会最明显提升“像不像”。
