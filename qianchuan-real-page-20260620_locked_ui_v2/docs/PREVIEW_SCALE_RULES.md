# Preview Scale Rules

## 固定设计画布

- 锁 UI 设计画布固定为 `2560px × 1348px`。
- 页面内部模块必须在固定画布内排布，不按浏览器窗口做响应式重排。
- 导出截图使用默认地址 `http://localhost:4173`，保持 `1:1` 原尺寸画布。

## 浏览器预览缩放

- 默认预览地址 `http://localhost:4173` 使用 `1:1` 画布，浏览器 100% 下需要滚动查看。
- 如需在普通浏览器窗口内整体看全，可访问 `http://localhost:4173/?previewScale=fit`。
- 预览缩放只允许作用在外层 `.preview-canvas`，不能分别缩放指标卡、趋势图或右侧卡片。

## 趋势图比例

- 趋势图 SVG 跟随固定画布中的固定趋势图容器。
- 浏览器 50%、100% 或 `previewScale=fit` 下，趋势图相对卡片的比例必须一致。
- 不允许为趋势图单独添加 `zoom` 或 `transform: scale()`。
