# UI Lock Rules

首版锁定目标是：千川页面截图-回放版本.png。

评论状态参考：千川页面截图-评论版.png。

局部细节参考：顶部导航栏局部截图.png、趋势图局部截图.png、右侧卡片局部截图.png。

reference-luopan-dashboard-v2.png.png 暂时只作为旧罗盘风格参考，不作为首版主视觉基准。

页面结构、固定中文文案、字体、间距、右上角状态区、表格样式、红字提示、卡片布局不得自由发挥。

没有截图或 DOM 依据的 UI，不允许猜。

缺少依据的地方必须记录为“缺少参考依据”。

后续实现时，UI 层只能消费数据，不能把业务数据硬编码进结构或样式。
