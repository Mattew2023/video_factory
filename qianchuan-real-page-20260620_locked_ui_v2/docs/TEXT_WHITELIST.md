浏览器插件、监控、安全、SDK、埋点相关文案不进入白名单。

## 明确不进入白名单

- 主播名，例如源 HTML 中的主播名。
- 日期、开播完整时间、直播时长。
- GMV、ROI、消耗、成交金额、曝光、观看、订单等数值。
- 带业务数量的提示文案，例如“存在2次素材调整”。此类文案需要拆成固定模板和数据字段，当前触发规则仍缺少参考依据。
- 浏览器插件、监控、安全、SDK、埋点相关文案。

## 缺少参考依据

- 红字提示的完整固定模板缺少参考依据。
- 表格页签、表头和行内文案缺少参考依据。
- 筛选弹层、指标配置弹层、数据口径说明弹层缺少参考依据。
