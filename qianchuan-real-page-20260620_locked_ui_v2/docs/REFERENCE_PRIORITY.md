## 使用规则

- 首版不得混用罗盘风格。
- `reference-luopan-dashboard-v2.png.png` 只用于理解旧罗盘风格差异，不作为首版主视觉基准。
- 当截图与 HTML DOM 存在冲突时，首版视觉以 P0/P1 截图为准，HTML DOM 只用于确认模块存在、固定文案和数据字段。
- 没有截图或 DOM 依据的 UI，不允许自行补全，必须写入 `UNVERIFIED_GAPS.md`。
- 后续第三阶段静态布局复刻只能从 P0/P1 和 HTML DOM 已确认的模块开始。
