# Phase 4 Metrics Notes

## 范围

- 本阶段只对齐核心指标卡区域。
- 未修改顶部导航逻辑、公告条逻辑、趋势图逻辑、右侧卡片逻辑、漏斗逻辑、评论/直播画面逻辑。

## 已调整

- 微调核心指标卡高度、圆角、边框和蓝色背景层次。
- 收紧核心指标卡与下方趋势图区之间的间距。
- 调整指标名称字号、指标数字字号、行高和整数/小数/百分号的紧密排布。
- 微调右上角“全域净成交数据”按钮和两个圆形图标的尺寸与位置。

## 缺少依据

- 右上角两个圆形图标的准确 icon 资源缺少可迁移依据，当前仍使用近似字符。
- 指标名称 tooltip 或 hover 态缺少展开截图依据，本阶段未实现。

## 4.1 缩放收口

- 已确认旧结构使用 `height: 100vh` 和小视口 media query，会导致不同浏览器缩放下不再是同一张固定设计画布。
- 已补充固定 `2560px × 1348px` 设计画布和外层 `.preview-canvas` 预览缩放规则。
- 默认地址保持 `1:1` 原尺寸，用于导出截图；`?previewScale=fit` 仅用于浏览器预览看全。
